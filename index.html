Welcome To Feisty Pass, the password encryption app!

Just getting started? It's strongly recommended you change a different salt in the config menu before beginning.

Learn more about Feisty Pass, salt, and encryption in general by seeing the help section.
What Is Feisty Pass?
Feisty Pass is a password management app that automatically encrypts its entries by using feistel networks and another password.
Will my passwords be safe?
Based on my understanding of cryptography, yes. However, I am not an expert in cryptography and can make no guarantee and thus can make no promises.
Creating a New Entry
  1. Press the "New" button at the lower right of the screen.
    This will display the form used to create new entries
  2. Optional: Give the entry a title and description.
    Though not required, they are intended to help you recognize and use your passwords. For example a website could require some special condition before or afer entering your password and a description is the perfect place to make note of it
  3. Optional: add and delete fields from the section that will be encrypted.
    Though for many things a username and password are enough, some things require more data that you don't want other people knowing. This is also a good place to put the answers to security questions.
    Add a feild by pressing the "More Rows" button. Delete a feild by pressing the "X" button to the left of the field.
  4. Optional: set a password.
    This is highly recommended, as without a password the only thing protecting the entry is the salt (see below).
  5. Press the "Create New Entry" button.
    This will create a new entry using the information you've filled in.
Editing an Entry
You can edit an entry even if it is encrypted. But if it is encrypted, you can only see and edit the title and description. Other then that, editing is the same as making a new entry; simply press the "Edit" button and change what you want.
Note: it is currently not possible to edit an entry's password to be blank. If the password and confirm feild are left blank, the entry's password doesn't change.
Selecting an Entry
The list of current entries is shown on the right side of the screen. To select an entry, simply click it.
Sorting Entries
The list of current entries is shown on the right side of the screen. To rearrange entries, you can drag then up and down with the mouse and place them where you like.
Deleting Entries
To delete an Entry, simply select the entry you want to delete, then press the "delete" button.
Encrypted and decrypted entries can be deleted. Once an entry is deleted it is gone for good, unless you've saved it to an external file.

The Following is a light overview of cryptography

What is salt?
A salt is a random string used to create additional protections against a variety of decryption methods.
For example, an attacker may take a list of common passwords and encrypt them to see if any match our encrypted passwords. By having a salt that's different then the default, the attacker can't see if the password matches any of the previous ones without having to do the same work all over again with the new salt.
What is a Feistel Network?

A Feistel Network is a key element of many encryption schemes. The steps a Feistel Netork performs are as follows:

  1. Break the text into a left block (Ln) and a right block (Rn).
  2. Copy the right block into the new left block.
    Ln+1 = Rn
  3. Run the old right block and a key through a one way function.
    x = F(Rn, Kn)
  4. Combind the previous result with the old left block to create the new right block.
    Rn+1 = Ln ⊕ x
  5. Repeat steps 1 through 4 for as many times as specified.

This process can be used to encrypt and decrypt a block of text.
